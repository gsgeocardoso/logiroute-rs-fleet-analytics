# RS Spatial Delivery H3: Análise Espacial e Desempenho Operacional de Entregas no RS

## Visão Geral
Projeto de Inteligência Geoespacial e Business Intelligence focado na análise de desempenho operacional e logística de e-commerce no Estado do Rio Grande do Sul.

O objetivo do projeto é estruturar um pipeline de dados capaz de mapear gargalos logísticos, tempos médios de entrega e taxas de atraso em nível microregional, utilizando indexação espacial hexagonal para otimizar a performance de visualização de dados.

## Tecnologias Utilizadas
- **Python (Pandas, H3):** ETL, tratamento de dados temporais/geográficos e indexação espacial em malha hexagonal Uber H3 (Resolução 7).
- **Power BI:** Modelagem de dados, mapas de densidade hexagonal e dashboard.
